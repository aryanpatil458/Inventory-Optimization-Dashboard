# Inventory-Optimization-Dashboard
This project focuses on analyzing retail inventory data to optimize stock levels, reduce stockouts, and minimize overstock situations. It uses Python, SQL, and data visualization techniques to generate actionable business insights.  The project simulates scenarios, where efficient inventory management is critical for profitability and operations.
